Ya Notepad

A plain text editor that just works

No spell check. No autocorrect. No distractions.

What It Does

Plain Text

Opens, edits, and saves plain text files. UTF-8, nothing else. What you type is what you get.

Multiple Windows

Open as many files as you need. Each window has its own document with independent undo history.

Monospace Font

Clean, consistent text with the system monospace font. Every character lines up.

Word Wrap

Long lines wrap at the window edge. Resize the window and the text reflows to match.

Undo & Redo

Full undo and redo support. Cmd+Z and Cmd+Shift+Z work exactly as you expect.

Unsaved Changes

Prompts before closing or opening over unsaved work. Your text won't disappear by accident.

What It Doesn't Do

Spell Check
Autocorrect
Grammar
Smart Quotes
Smart Dashes
Link Detection
Data Detection
Text Completion
Rich Text
Markdown
AI Anything
Format Menu

Why

Every “smart” feature is code running against your input. Every Markdown renderer is an attack surface. Every AI integration is a network call with your data.

Ya Notepad has none of that. It's a native macOS app built with SwiftUI wrapping a plain NSTextView with every smart feature explicitly turned off. No Electron, no web views, no bundled runtime.

Sometimes the best feature is the absence of features.
