Features

Unified Dashboard

Monitor your entire media stack from a single app. TV, movies, music, downloads, and requests at a glance.

Multi-Server

Connect to all your services at once. Switch between Sonarr, Radarr, Lidarr, and more with a tap.

NZB Search

Search Newznab indexers directly. Browse results with poster thumbnails, manage a cart, and save RSS feeds.

Download Management

Full control over SABnzbd, qBittorrent, and Transmission. Pause, resume, prioritize, and monitor in real-time.

Calendar & Queue

See upcoming episodes and album releases. Track what's downloading and importing across all services.

Media Requests

View and manage Overseerr/Jellyseerr requests. Approve, decline, and search for new content to add.

Siri Shortcuts

Check downloads, browse upcoming releases, and control your server hands-free with built-in Siri Shortcuts.

SSH & SFTP

Browse remote filesystems, manage files, and run commands on your server directly from the app.

Secure

API keys stored in iOS Keychain. Supports HTTPS, API key auth, basic auth, and SSH key authentication.
